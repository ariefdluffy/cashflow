# Cash Flow Dashboard

Dashboard monitoring arus kas real-time dari Google Sheets.

## Quick Start

```bash
bun install
bun run dev
```

## Build for Production

```bash
bun run build
bun run preview
```

## Deploy with PM2

```bash
pm2 start ecosystem.config.cjs
```

## Tech Stack

- Svelte 5 + SvelteKit
- TypeScript
- Tailwind CSS v4
- Chart.js
- Bun Runtime
- Google Sheets API
